## 计算圆周率

```bash
Estimating Pi with Rust...
Rust Pi Calculation: 241.897ms
Estimated Pi (Rust): 3.14162672
Estimating Pi with JavaScript...

```

## 总结

rust在不同场景速度皆优于js，在快排侧速度快1倍，在计算圆周率侧要快6倍。
